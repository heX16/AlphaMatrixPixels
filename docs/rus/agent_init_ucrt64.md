# Инструкция для AI агента: Инициализация UCRT64

**AI агент**: Перед выполнением _первой_ команды выполни особую команду инициализации.

Не делай слияния команды инициализации и другой команды. Пиши команду инициализации и смотри результат выполнения.

**Важно**: Инициализацию нужно выполнять только **один раз в самом начале сессии**, а не перед каждой командой. После этой инициализации, все команды нужно выполнять напрямую.

**Важно**: не запускай `bash`! Ты уже работаешь в bash! Можно писать команды напрямую, без приставки `bash`.


Команда инициализации `bash` консоли:
```bash
export MSYSTEM=UCRT64 && export MSYS2_PATH_TYPE=inherit && ([ -f /ucrt64/etc/profile ] && source /ucrt64/etc/profile || source /etc/profile)
```
