# Инструкция для AI агента: Инициализация UCRT64

**AI агент**: Перед выполнением первой команды в терминале в начале каждой сессии выполни:

```bash
export MSYSTEM=UCRT64 && export MSYS2_PATH_TYPE=inherit && ([ -f /ucrt64/etc/profile ] && source /ucrt64/etc/profile || source /etc/profile)
```

**Важно**: Инициализацию нужно выполнять только **один раз в самом начале сессии**, а не перед каждой командой. После первой инициализации можно выполнять команды напрямую.
