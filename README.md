# AlphaMatrixPixels

Библиотека для работы с пикселями в памяти — **с альфа‑каналом и корректным смешиванием цветов**.

## Что умеет

- **2D-канва** — хранение изображения, чтение и запись пикселей по координатам
- **Прозрачность** — 8-битный альфа‑канал в каждом пикселе
- **Смешивание** — наложение цветов "как в фотошопе", с поддержкой глобальной прозрачности
- **Наложение изображений** — рисование одной картинки поверх другой
